# How to Build
1. Clone this repository.
2. Create an `libs` folder inside the project.
3. Clone Dependencies into it:
   `git clone https://github.com/libsdl-org/SDL libs/SDL`
   `git clone https://github.com/gabime/spdlog.git libs/spdlog`
4. Run the Emulator
    `.launch.sh`

# Dependency List
* [SDL3](https://www.libsdl.org/)
* [spdlog](https://github.com/gabime/spdlog.git)
